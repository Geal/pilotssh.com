About

PilotSSH is a secure application for server administration.

It uses a set of scripts on the server side to create a nice graphical user interface. The scripts are all open source and free to use, and available on Github.

  • WordPress admin

  • Process list

    Display the list of running process, and kill one of them when touched

  • Apache sites

    Show which sites are enabled in Apache. Enable and disable them with a touch

  • Uptime

    Display the uptime. Shut down and reboot the server
